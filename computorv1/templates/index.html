Help

This is Computor V1 for the programming school 42
The project is about resolving polynomial equation until degree 2 !
You can get read the french subject

How to

Enter some equation in the two fields in the main page :
5 * X^0 + 4 * X^1 - 9.3 * X^2 = 0

or something more natural like :
5 + 4x - 9,3 x**2

When only one field is filled, the second one takes 0 by default.


Bonus

Steps Input and Formatted
4 -> 4 * X^0
x -> 1 * X^1
5x -> 5 * X^1
x**2 -> X^2
x * 5 -> 5 * X^1
replacing spaces
00.0 -> 0
multi coma : , .
01 -> 1
fraction
colorize in terminal (-g)
raise in terminal (-e)
